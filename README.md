randomplate
===========

Random Plate Number Generator

A tool to generate random plate numbers together with name of drivers created using Apache Cordova.  (Android 4.0 and higher recommended)

To run the application:

1. Install Apache Cordova CLI: 
	http://cordova.apache.org/docs/en/4.0.0/guide_cli_index.md.html
2. On the project root directory, open a terminal/command prompt and run:
	cordova run
    (You must have a running android virtual device.)
